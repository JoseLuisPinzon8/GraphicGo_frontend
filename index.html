GraphicGo

GraphicGo es un visualizador de código del lenguaje Chocopy. Simplemente escribe el código en la caja de abajo y presiona el botón Ver Ejecución. La visualización de tu código se encontrará en la sección Resultados.

Demostración

A continuación te mostramos un ejemplo de lo que puedes hacer con GraphicGo. Copia y pega el código de abajo.

Ejemplo:
def a()->str:
    for y in [1,2,3,4]:
        print(y)
        if(y > 2):
            return "funciona"
    return "no funciona"

print(a())
